== Description ==

Magic Shortcodes is a very unique and powerful "Shortcode Builder". You can easily create shortcodes from any HTML/CSS/JS code. You can also style and configure existing shortocdes. All Shortcodes are dynamically generated and you can easily add them to post and pages and there is also a widget available for you to easily add them to widget area.

== Installation ==

1. Upload `magic-shortcodes` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Activated menu can be found in left side menubar.
4- Add Magic Shortcodes to display in your website.
